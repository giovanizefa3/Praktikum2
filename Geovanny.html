1.1 Sejarah HTML

HTML adalah bahasa dasar untuk menampilkan halaman web pada web browser. HTML merupakan singaktan dari Hypertext Markup Language. Yang artinya adalah bahasa markup (penanda) berbasis text atau bisa disebut sebagai formatting language. HTML bukanlah bahasa pemprograman tetapi bahasa markup formatting.

HTML dibuat oleh Tim Berners Lee ketika masuk bekerja untuk CERN dan dipopulerkan pertama kali oleh browse Mosaic. selama awal tahun 1990 HTML mengalami perkembangan yang sangat pesat. Setiap pengembangan HTML pasti akan menambahkan kemampuan dan fasilitas yang baik dari versi sebelumnya. Sebelum suatu HTML disahkan sebagai suatu dokumen HTML Standar, ia harus disetujui dulu oleh W3C untuk dievaluasi secara ketat. Setiap terjadi perkembangan suatu versi HTML, maka mau tak mau browser pun harus memperbaiki diri agar bisa mendukung kode - kode HTML yang baru tersebut. Sebab jika tidak, browse tak akan bisa menampilkan HTML tersebut

saat ini HTML sudah mencapai versi 5 atau bisa disebut HTML5. Versi ini memiliki banyak fitur yang dapat digunakan dan memiliki beberapa tambahan penggunaan script pada editor.

HTML biasa ditulis pada sebuah text editor (dreamweaver, notepad++, notepad, phpstorm, brackets, netbeans, dll). ketik kutipan coding HTML diatas pada texteditor anda dan kemudian simpan dengan format extension. html yang menandakan bahwa file tersebut adalah file HTML
